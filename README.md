# 🍎 Sistema de Control y Gestión de Desnutrición Infantil
**Child Malnutrition Control and Management System**
 
> Plataforma web para el monitoreo y control nutricional infantil, orientada a facilitar el seguimiento de indicadores de salud en poblaciones vulnerables.
>
> Web platform for child nutritional monitoring and control, designed to facilitate health indicator tracking in vulnerable populations.

## 📋 Descripción / Description
 
**ES:** Sistema web desarrollado como proyecto de graduación, que permite registrar, dar seguimiento y visualizar indicadores nutricionales infantiles. El sistema facilita la gestión eficiente de información para apoyar la toma de decisiones en contextos de salud pública.
 
**EN:** Web system developed as a graduation project, allowing the registration, tracking, and visualization of child nutritional indicators. The system enables efficient information management to support decision-making in public health contexts.
 
---
 
## ✨ Funcionalidades / Features
 
- Registro y gestión de pacientes / Patient registration and management
- Monitoreo de indicadores nutricionales / Nutritional indicator monitoring
- Visualización de datos y reportes / Data visualization and reports
- Interfaz intuitiva y responsiva / Intuitive and responsive interface
- API REST para integración / REST API for integration

## 🚀 Instalación local / Local Setup
 
```bash
# Clonar el repositorio / Clone the repository
git clone https://github.com/SaintXini/<nombre-del-repo>
cd <nombre-del-repo>
 
# Backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
 
# Frontend
cd frontend
npm install
npm run dev
```
